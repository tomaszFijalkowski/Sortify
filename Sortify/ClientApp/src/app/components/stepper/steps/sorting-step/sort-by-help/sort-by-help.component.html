Sort by

Fill this section with track properties you want to sort by.

The importance of properties goes from left to right and is indicated by the ordinal number. If two tracks have equal values on one property (e.g. same album or artist), the next property from the list will be considered.

You can set the sorting to be ascending arrow_upward or descending arrow_downward.

Not sure where to start? Click the star icon for a recommended sorting.
